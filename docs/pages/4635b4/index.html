# 1、下载配置 ingress-nginx 文件

# 使用 LoadBalancer
# 适合于没有 Helm 的环境，或者是倾向于使用 YAML 配置
wget https://jihulab.com/mirrors-github/kubernetes/ingress-nginx/-/raw/controller-v$INGRESS_NGINX_VERSION/deploy/static/provider/cloud/deploy.yaml
